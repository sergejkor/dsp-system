Production-leaning SVG overlay pack for three vehicle families:
- sprinter
- boxer
- rivian

Each folder contains 8 portrait overlays (1080x1920) with transparent background.

Suggested mapping:
- front_left
- left_side
- rear_left
- rear
- rear_right
- right_side
- front_right
- front

Use on top of the live camera preview with pointer-events:none and ~0.55 opacity.
